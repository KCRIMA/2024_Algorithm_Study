# 2024-Algorithm_Study

### ☑️스터디 규칙 
- 일주일에 한번 회의 시간 (주차마다 한명 지정)
- 일주일에 문제 3번 이상 풀고 PUSH
- Merge Pull Request요청은 스터디 회의 시작 시각에 승인
- 승인 이후 동기화 작업
  
### ☑️문제 풀이
- 코딩마스터즈
- 백준
- 프로그래머스<br>
-양식 상관 없이 본인 이름_해당 날짜 폴더에 업로드
